INSTALLATION INSTRUCTIONS
--------------------------

Method 1 (Easiest):
- Download .fb2k-component files
- Double-click each file
- foobar2000 will install automatically

Method 2 (Manual):
- Download .fb2k-component files
- Drag and drop into foobar2000 window
- Restart foobar2000

Method 3 (Advanced):
- Extract .fb2k-component files (they're ZIP archives)
- Copy .dll files to: C:\Program Files\foobar2000\components\
- Restart foobar2000


VERIFICATION
------------

After installation, verify components are loaded:
1. File > Preferences > Components
2. Check that all installed components appear in the list
3. Look for any error messages


TROUBLESHOOTING
----------------

Component won't install:
- Make sure foobar2000 is closed
- Run installer as administrator
- Check component compatibility with your foobar2000 version

Component not appearing:
- Restart foobar2000
- Check Preferences > Components for errors
- Reinstall the component

Crashes after installing component:
- Remove the component from the components folder
- Update foobar2000 to latest version
- Try an older version of the component
