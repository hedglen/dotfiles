# Premium Gold Executive Theme - Quick Setup Guide

Get your foobar2000 looking premium in 10 minutes.

## Step 1: Open Georgia-Reborn Settings

1. Right-click anywhere in foobar2000
2. Select **Options** → **Georgia-Reborn**

## Step 2: Apply Color Scheme

### Theme Tab
- **Theme:** Custom
- **Theme Preset:** Dark
- **Primary Color:** `#D4AF37` (Gold)
- **Accent Color:** `#FFD700` (Bright Gold)

### Colors Tab

Copy these exact values:

**Background:**
- Main: `#0D0D0D`
- Panel: `#121212`
- Sidebar: `#1A1A1A`
- Player Bar: `#1E1E1E`

**Text:**
- Primary: `#F5F5DC`
- Secondary: `#C9B037`
- Highlight: `#FFD700`
- Playing Track: `#F4C430`

**Accents:**
- Primary: `#D4AF37`
- Border: `#3D3D3D`
- Selection BG: `#2A2A2A`
- Selection Text: `#FFD700`

## Step 3: Configure Waveform

1. Go to **Preferences** → **Tools** → **ESLyric**
2. Click **Waveform** tab
3. Enable "Show waveform seekbar"
4. Set colors:
   - Background: `#121212`
   - Waveform: `#D4AF37`
   - Progress: `#FFD700`
   - Cursor: `#F4C430`
5. Style: **Filled**
6. Enable **Peak Hold**

## Step 4: Setup Spectrum Analyzer

1. Right-click on spectrum area
2. Select **Settings**
3. Configure:
   - Bars: **32**
   - Smoothing: **High**
   - Colors:
     - Low: `#8B7355` (Bronze)
     - Mid: `#D4AF37` (Gold)
     - High: `#FFD700` (Bright Gold)
     - Peak: `#F4C430`
   - Enable **Glow** (15% intensity)
   - Enable **Reflection** (25% opacity)

## Step 5: Configure Equalizer

1. **Preferences** → **Playback** → **DSP Manager**
2. Add **Equalizer**
3. Apply "Warm Executive" preset:
   - 32 Hz: +2 dB
   - 64 Hz: +1 dB
   - 1 kHz: +1 dB
   - 2 kHz: +2 dB
   - 4 kHz: +1 dB
   - 16 kHz: +1 dB
4. Enable **Show frequency response curve**
5. Curve color: `#D4AF37`

## Step 6: Adjust Layout

### Recommended Panel Arrangement:

**Top:** Player bar (80px height)
- Album art (left)
- Track info (center)
- Controls (right)

**Middle:** 3-column layout
- Library (25% width)
- Playlist (50% width)
- Details/Lyrics (25% width)

**Bottom:** Visualizers (200px height)
- Waveform seekbar (top)
- Spectrum analyzer (bottom)

## Step 7: Typography

**Preferences** → **Display** → **Georgia-Reborn** → **Typography**

- Primary Font: **Segoe UI Semibold**
- Track Title: **11pt**
- Artist/Album: **10pt**
- Playlist: **9pt**

## Step 8: Album Art

**Preferences** → **Display** → **Album Art**

- Size: **300x300px** minimum
- Border: **2px** solid `#3D3D3D`
- Shadow: **Enabled** (gold glow)
- Reflection: **40%** opacity

## Step 9: Final Touches

1. **Enable smooth scrolling**
2. **Set animation duration:** 200ms
3. **Enable font smoothing** (Windows settings)
4. **Adjust monitor contrast** for optimal gold visibility

## Step 10: Save & Test

1. Save theme as **"Premium Gold Executive"**
2. Play a track with good album art
3. Verify:
   - ✓ Waveform displays in gold
   - ✓ Spectrum animates smoothly
   - ✓ Colors look rich and premium
   - ✓ Text is readable
   - ✓ Album art has gold glow

## Keyboard Shortcuts

Set these up for quick access:

- `Ctrl+E` - Toggle Equalizer
- `Ctrl+V` - Toggle Visualizer
- `Ctrl+L` - Toggle Lyrics
- `F11` - Fullscreen

## Troubleshooting

**Gold looks washed out?**
- Increase monitor contrast
- Check color profile (use sRGB)
- Adjust room lighting

**Visualizers laggy?**
- Reduce spectrum bars to 24
- Lower update rate to 30 FPS
- Disable reflection effects

**Text hard to read?**
- Increase font size by 1pt
- Enable ClearType in Windows
- Adjust text color brightness
