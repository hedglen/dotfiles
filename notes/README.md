# Notes

Personal markdown notes (tech, workflow, general). This folder lives **inside [dotfiles](https://github.com/hedglen/dotfiles)** at `dotfiles/notes/` — commit with the rest of your workstation config, or keep sensitive paths out of the public remote via branch/private fork as you prefer.

Machine layout, apps, and bootstrap: **`dotfiles/docs/workstation-setup.md`**.

## Structure

```text
notes/
├── personal/   personal notes
├── tech/       technical references and how-tos
└── work/       work-related notes
```
