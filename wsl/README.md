- `.p10k.zsh`

These are meant to mirror the live files in WSL:

- `/home/rjh/.zshrc`
- `/home/rjh/.p10k.zsh`

Current setup:

- `oh-my-zsh`
- `powerlevel10k`
- classic powerline-style prompt with practical segments

If the WSL prompt changes, update the live files in WSL and then copy the changes back here.
